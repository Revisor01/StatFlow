## Notes for Review

- App requires a self-hosted Umami or Plausible Analytics server to function
- Test accounts above provide real analytics data for review purposes
- No user registration in the app itself — users enter their own server credentials
- App does not collect any user data (see privacy policy: https://simonluthe.de/apps/statflow/datenschutz/)
- The app communicates exclusively with the configured analytics servers — no other network connections are made
- Widget requires adding the widget from the iOS widget gallery after signing in

## App Functionality Overview

1. **Dashboard:** Shows all websites with key metrics (visitors, pageviews, bounce rate, session duration) and sparkline charts
2. **Website Detail:** Tap any website for detailed stats including top pages, referrers, devices, browsers, and geography
3. **Realtime:** Live visitor count and current page views
4. **Compare:** Side-by-side period comparison (e.g., this week vs. last week)
5. **Events & Reports:** Custom event tracking, funnel analysis, UTM parameters, goals, and attribution
6. **Widget:** Home screen widget showing key metrics for a selected website
7. **Settings:** Account management, notification preferences, dashboard customization
