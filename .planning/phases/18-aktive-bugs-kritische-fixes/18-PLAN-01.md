For each of the following 12 ViewModels, apply the canonical Task-Cancellation pattern from `WebsiteDetailViewModel`:

**Pattern to apply in each ViewModel:**

1. Add property: `private var loadingTask: Task<Void, Never>?`
2. For each public async method that loads data, wrap with:
   ```swift
   func loadX(...) async {
       loadingTask?.cancel()
       let task = Task {
           // existing body, plus:
           // defer { if !Task.isCancelled { isLoading = false } }  (if isLoading exists)
           // guard !Task.isCancelled else { return }  after every await before @Published writes
       }
       loadingTask = task
       await task.value
   }
   ```
3. Add `func cancelLoading() { loadingTask?.cancel(); loadingTask = nil }` if the ViewModel has a view with `.onDisappear`

**ViewModels to fix (file, class name, async methods to wrap):**
